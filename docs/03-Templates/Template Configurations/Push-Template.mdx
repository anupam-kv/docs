# Push Notification Templates

Push notification templates in Siren allow you to create reusable notification formats for mobile devices. These templates support:

- Customizable notification title and body
- Platform-specific configurations
- Mobile preview options
- Image and icon URLs
- Time-to-live (TTL) settings

## Creating Push Templates

1. Navigate to the Templates section
2. Click "Create New Template"
3. Select Push Notification channel
4. Configure the following:
   - Notification title
   - Notification body
   - Platform-specific settings
   - Mobile preview options
   - Image and icon URLs
   - TTL settings

## Best Practices

- Keep messages concise and impactful
- Use platform-specific optimizations
- Include relevant icons and images
- Set appropriate TTL values
- Test on different devices
