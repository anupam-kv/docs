# SMS Templates

SMS templates in Siren allow you to create reusable SMS message formats that can be used across different workflows. These templates support:

- Text content with up to 160 characters
- Unicode support
- Flash message option
- DLT template ID configuration

## Creating SMS Templates

1. Navigate to the Templates section
2. Click "Create New Template"
3. Select SMS channel
4. Configure the following:
   - Message content
   - Unicode option (for special characters)
   - Flash message toggle
   - DLT template ID (if applicable)

## Best Practices

- Keep messages concise and clear
- Use variables for personalization
- Test with different character sets
- Follow carrier guidelines for message length
