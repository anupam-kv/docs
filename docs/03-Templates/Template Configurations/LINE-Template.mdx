# LINE Templates

LINE templates in Siren allow you to create reusable message formats for LINE integration. These templates support:

- Rich message content
- LINE-specific formatting
- Interactive elements
- Media attachments

## Creating LINE Templates

1. Navigate to the Templates section
2. Click "Create New Template"
3. Select LINE channel
4. Configure the following:
   - Message content
   - LINE-specific formatting
   - Interactive elements
   - Media attachments

## Best Practices

- Follow LINE message guidelines
- Use appropriate formatting
- Include necessary media
- Test with LINE integration
