# Email Templates

Email templates in Siren allow you to create reusable email formats with rich HTML content and attachments. These templates support:

- HTML content with custom styling
- Email subject line
- Attachments
- Advanced email headers
- Branding elements

## Creating Email Templates

1. Navigate to the Templates section
2. Click "Create New Template"
3. Select Email channel
4. Configure the following:
   - Subject line
   - HTML body content
   - Attachments (optional)
   - Custom email headers
   - Branding elements

## Best Practices

- Use clear and descriptive subject lines
- Include branding elements
- Use responsive HTML for mobile compatibility
- Add alt text for images
- Follow email client compatibility guidelines
