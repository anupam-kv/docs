# Discord Templates

Discord templates in Siren allow you to create reusable message formats for Discord server integration. These templates support:

- Rich embed messages
- Channel targeting
- Discord-specific formatting
- Interactive elements

## Creating Discord Templates

1. Navigate to the Templates section
2. Click "Create New Template"
3. Select Discord channel
4. Configure the following:
   - Embed messages
   - Channel targeting
   - Discord-specific formatting
   - Interactive elements

## Best Practices

- Use appropriate embed layouts
- Target specific channels
- Include interactive elements
- Follow Discord message guidelines
