# Slack Templates

Slack templates in Siren allow you to create reusable message formats for Slack workspace integration. These templates support:

- Rich message blocks
- Channel targeting
- Interactive elements
- Slack-specific formatting

## Creating Slack Templates

1. Navigate to the Templates section
2. Click "Create New Template"
3. Select Slack channel
4. Configure the following:
   - Message blocks
   - Channel targeting
   - Interactive elements
   - Slack-specific formatting

## Best Practices

- Use appropriate message blocks
- Target specific channels
- Include interactive elements
- Follow Slack message guidelines
