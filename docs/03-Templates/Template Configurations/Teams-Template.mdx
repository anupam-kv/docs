# Microsoft Teams Templates

Microsoft Teams templates in Siren allow you to create reusable message formats for Teams integration. These templates support:

- Rich message cards
- Channel targeting
- Adaptive cards
- Teams-specific formatting

## Creating Teams Templates

1. Navigate to the Templates section
2. Click "Create New Template"
3. Select Microsoft Teams channel
4. Configure the following:
   - Message cards
   - Channel targeting
   - Adaptive cards
   - Teams-specific formatting

## Best Practices

- Use appropriate card layouts
- Target specific channels
- Include adaptive formatting
- Follow Teams message guidelines
