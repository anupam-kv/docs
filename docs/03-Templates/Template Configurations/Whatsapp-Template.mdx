# WhatsApp Templates

WhatsApp templates in Siren allow you to create reusable message formats for WhatsApp Business API integration. These templates support:

- Text messages with rich media
- WhatsApp Business API compliance
- Message templates for different use cases
- Media attachments

## Creating WhatsApp Templates

1. Navigate to the Templates section
2. Click "Create New Template"
3. Select WhatsApp channel
4. Configure the following:
   - Message content
   - Media attachments (optional)
   - WhatsApp Business API settings
   - Template approval status

## Best Practices

- Follow WhatsApp Business API guidelines
- Use approved message templates
- Include necessary media content
- Test with WhatsApp Business API
