# In-App Templates

In-App templates in Siren allow you to create reusable message formats for in-application notifications. These templates support:

- Customizable message layouts
- App-specific targeting
- Rich media content
- Interactive elements

## Creating In-App Templates

1. Navigate to the Templates section
2. Click "Create New Template"
3. Select In-App channel
4. Configure the following:
   - Message layout
   - App targeting
   - Rich media content
   - Interactive elements

## Best Practices

- Use appropriate message layouts
- Target specific app sections
- Include rich media content
- Follow app-specific guidelines
