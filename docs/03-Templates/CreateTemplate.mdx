# Creating Templates

To create a new template in Siren:

1. Navigate to the Templates section
2. Click "Create New Template"
3. Choose the channels you want to configure
4. Fill in the template details:
   - Template name
   - Template description
   - Tags (optional)
5. Configure each channel's content
6. Click "Save Template"

## Template Fields

- **Template Name:** A unique name for your template
- **Description:** Brief description of the template's purpose
- **Tags:** Optional tags to categorize the template
- **Channels:** Select one or more channels to configure

## Best Practices

- Use clear and descriptive names
- Add helpful descriptions
- Use relevant tags for organization
- Test templates before production use
- Document any special requirements
