# Template Configuration

Template configuration in Siren allows you to:

- Edit template content
- Update template settings
- Manage template versions
- Configure channel-specific settings

## Editing Template Content

1. Select a template from the list
2. Click "Edit Template"
3. Make your changes
4. Click "Save Changes"

## Template Settings

- **Status:** Active/Inactive
- **Tags:** Add or remove tags
- **Description:** Update template description
- **Variables:** Manage dynamic content variables

## Version Management

- View template history
- Restore previous versions
- Compare versions
- Track changes

## Best Practices

- Regularly update descriptions
- Use version control
- Document changes
- Test updates before saving
