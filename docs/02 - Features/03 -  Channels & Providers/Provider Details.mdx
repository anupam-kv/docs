# Provider Details

Each channel is powered by one or more providers—external services that handle the delivery of messages like emails, texts, or push notifications.  
Siren supports over 20 provider integrations and allows multiple providers per channel to enable fallback mechanisms and intelligent routing.


**Examples of Supported Providers:**
- **Twilio** – Popular for SMS and voice messaging.  
- **MessageBird** – Omnichannel communication including SMS, WhatsApp, and voice.  
- **SendGrid** – Scalable email delivery service with high deliverability rates.

## Provider Configuration

You can:

- Configure provider credentials
- Manage priority and fallback settings
- Monitor performance directly from the dashboard

Each integration includes:

- Comprehensive setup instructions
- Environment support for both Testing** and Production modes
