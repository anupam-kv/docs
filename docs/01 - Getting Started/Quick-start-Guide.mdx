## Quick Start Guide

Set up your Siren account and start sending messages in just a few steps.

---

### Step 1: [Sign Up to Siren](https://app.trysiren.io/sign-up)

Create your Siren account to access the dashboard, manage workflows, and configure communication channels.

---

### Step 2: Integrate the SDK and MCP Server

#### SDK Integration

Use the Siren SDK to trigger workflows and manage communication directly from your backend.\
[Read the SDK Documentation](https://docs.trysiren.io/docs/SDK)

#### MCP Server Setup

Integrate the MCP (Model Context Protocol) server to allow AI agents to automatically handle workflow triggers and integrations.\
[Learn about MCP Integration](https://docs.trysiren.io/docs/MCP%20Server)

#### AI Agent Toolkit Integration

---

### Step 3: Generate API Key and Token

After logging in:

1. Go to the **right panel** → **Settings** → **Configurations**.
2. Click on **"Add API Key"**.
3. Enter a name for your API key.
4. An **API Token** will be generated.

> **_Note_**: Save this token securely — it is required for authenticating all API requests.

[API Reference](https://docs.trysiren.io/docs/API%20Reference)

---

### Step 4: Set Up Providers

1. In the **left panel**, go to **Settings** → **Providers**.
2. You'll see available **channels** (SMS, Email, Push, etc.).
3. Under each channel, select a **provider** and click **"Add Account"**.
4. Enter the required credentials or API details.

> **_Note_**: You can add up to **5 accounts** across all providers.

[View Provider Setup Guide](https://docs.trysiren.io/docs/Providers)

---

You're now ready to create workflows and start communicating with your users\!\
